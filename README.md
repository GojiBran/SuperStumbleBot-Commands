# SuperStumbleBot-Commands
### Super Stumble Bot Commands

A "comprehensive" list of "all" the commands for **[Super Stumble Bot](https://github.com/GojiBran/SuperStumbleBot)**.

---

## **StumbleBot Commands**

### **General Commands**
| Command               | Description                                                                 |
|-----------------------|-----------------------------------------------------------------------------|
| `.commands`           | Lists all available commands.                                               |
| `.bot`                | Provides links to download the bot and its YouTube-only version.            |
| `.choose [options]`   | Randomly selects one of the provided options (e.g., `.choose apple, orange`).|
| `.self`               | Displays the user's information (nickname, handle, username, mod status).   |

### **Utility Commands**
| Command               | Description                                                                 |
|-----------------------|-----------------------------------------------------------------------------|
| `.time [timezone]`    | Displays the current time in the specified timezone (e.g., `.time pst`).    |
| `.zones`              | Lists all supported time zones.                                             |
| `.currency [amount] [from] to [to]` | Converts currency (e.g., `.currency 50 usd to dkk`).               |
| `.calc [expression]`  | Performs a calculation (e.g., `.calc 2+2`).                                 |
| `.convert [value] [unit] to [unit]` | Converts units (e.g., `.convert 10 cm to inches`).                 |
| `.roll [dice]`        | Rolls dice (e.g., `.roll 2d6` rolls two six-sided dice).                    |
